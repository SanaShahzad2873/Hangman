# Hangman Game in C++

This is a **basic Hangman game** built in C++ for beginners.  
Players try to guess a hidden word by entering one letter at a time.  
The game shows correct guesses and tracks wrong attempts until the word is guessed or the player loses.

## Features
- Simple console-based game
- Beginner-friendly and easy to understand
- Uses basic C++ concepts: loops, arrays, and conditional statements
- Randomly selects a word from a small word list

## How to Play
1. Compile the program using a C++ compiler.
2. Run the executable.
3. Enter letters to guess the word.
4. Try to guess the word before reaching the maximum number of wrong attempts.

## Skills Practiced
- C++ basics
- Loops and conditionals
- Arrays and strings
- Problem-solving and logic
